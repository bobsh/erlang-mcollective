About
=====

An erlang client for Marionette Collective (mcollective).

Installation
============

Prerequisites
-------------
* erlang R14A
  - kernel, stdlib, sasl
* erlang-yaml (from http://github.com/bobsh/erlang-yaml)

Compilation
-----------

Build the source code by running::

  make

To clean::

  make clean
